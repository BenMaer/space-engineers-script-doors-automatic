** Required values **
Programming block name: Programmable block- Door- Automatic Open/Close

* name suffix:
1 door:
[room door is facing] -> [room door is not facing]

2 doors:
[either room] <-> [the other room]

1 door Example:
Downstairs -> Reactor Room

2 doors Example:
Reactor Room <-> Outside

* door name:
Sliding Door- [name suffix]
1 door Example:
Sliding Door- Downstairs -> Reactor Room

2 doors Example:
Example:
1) Sliding Door- Reactor Room -> Outside
2) Sliding Door- Outside -> Reactor Room

* timer block name:
Timer Block- Automatic Doors- Close- [name suffix]
1 door Example:
Timer Block- Automatic Doors- Close- Downstairs -> Reactor Room

2 doors Example:
Example:
Timer Block- Automatic Doors- Close- Reactor Room <-> Outside

** Optional values **

* panel1 name:
Button Panel- [door name]
1 door Example:
Button Panel- Sliding Door- Downstairs -> Reactor Room

2 doors Example:
Example:
Button Panel- Sliding Door- Reactor Room -> Outside

* panel2 name:
Button Panel- [door name, but with suffix reversed]
1 door Example:
Button Panel- Sliding Door- Reactor Room -> Downstairs

2 doors Example:
Example:
Button Panel- Sliding Door- Outside -> Reactor Room


** SETUP **

* Timer block needs:
 - Call programming block, with argument:
`d:`[door name]+` a:Close`
1 door Example:
d:Sliding Door- Downstairs -> Reactor Room a:Close

2 doors Example:
Example:
d:Sliding Door- Reactor Room -> Outside d:Sliding Door- Outside -> Reactor Room a:Close

* Panel needs:
 - Call programming block, with argument:
`d:`[door name]+` a:Open t:`[timer block name]
1 door Example:
d:Sliding Door- Downstairs -> Reactor Room a:Open t:Timer Block- Automatic Doors- Close- Downstairs -> Reactor Room

2 doors Example:
Example:
d:Sliding Door- Reactor Room -> Outside d:Sliding Door- Outside -> Reactor Room a:Open t:Timer Block- Automatic Doors- Close- Reactor Room <-> Outside
